Project Kenworth

USU Power Electronics Laboratory

This project involves developing and testing a 1 megawatt wireless charging system for the Kenworth truck company. By implementing a charging pad in a parking spot and an electric semi truck, the trucks can be charged wirelessly.

Project UCSD

USU Power Electronics Laboratory

The end goal of this project is a product that can help recycle used car batteries. This product repeatedly charges and discharges different cells within a battery pack at different rates to normalize the health of all the cells, thus creating a battery pack ready for a different use.
